## HotXposed:

[![](https://jitpack.io/v/githubwing/HotXposed.svg)](https://jitpack.io/#githubwing/HotXposed)

## Useage


compile the library in your build.gralde

```
gradle

allprojects {
    repositories {
        jcenter()
        maven { url "https://jitpack.io" }
    }
}

dependencies {
	        compile 'com.github.githubwing:HotXposed:v1.0.0'
	}

```
